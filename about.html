Super Tuner
About

NOTE: The Super Tuner for Spotify is an open-source project, which means anyone is free to deploy their own versions of it wherever they want. This page was written by Rich Abdill, the author of the app, for the original deployment of the Super Tuner at richabdill.com. You may be reading it somewhere else, but odds are Rich has no idea it's there. Please don't blame him if this crops up somewhere weird.

What is this?

The Super Tuner is a search tool for Spotify that allows users to specify detailed parameters about what they're looking for—details Spotify knows about, but doesn't make available in their standard player.

Who are you?

My name is Rich Abdill. I'm a software developer, and I listen to Spotify for hours a day.

I am mostly interested in backend development, but this project seemed like a fun excuse to tinker with an interesting interface. I don't work for Spotify, and this tool is pretty much finished; I don't anticipate adding more features.

Did Spotify make this app?

No. The only affiliation this application has with Spotify is that it uses their API, explained below.

How does this work?

The Super Tuner uses the Spotify Web API, which is an online service that Spotify has made available for developers who want to create new tools for the Spotify ecosystem.

In particular, the app leverages Spotify's huge collection of track metadata: information about songs (tempo, key, loudness, even how "happy" it sounds) that is available via the API, but doesn't show up anywhere in the app itself.

This project originally started out as a way for me to practice developing in Golang, a language I currently use at my day job. However, I quickly realized that the whole thing could be done in client-side Javascript and I could save myself a lot of headache: Because the Super Tuner operates entirely in your browser, I don't have to bother with running servers and managing complicated software compilation and release processes—I just make sure the latest files are available, and your computer does the rest.

If you're interested in the technical nuts and bolts, check out the project on GitHub.

Are you stealing my data?

No! I can confidently say that I am not. The Super Tuner doesn't need to store anything about you for it to work. This is between you and Spotify: Whenever you click a button that sends a request—searching for artists, exporting a playlist, etc.—that request is sent directly from your computer to the Spotify servers. In fact, I don't even have a way of knowing you've used the Super Tuner at all.

Privacy policy

The Spotify Developer Terms of Use require me to publish a privacy policy with this app. This is the policy.

  • There is no user data collected by the Super Tuner. Any data incidentally contained in server logs is available only to administrators and will be deleted on a regular basis. (The only Spotify-specific information that is likely to appear in logs is the user token included in the URL of the page after you have logged in. These expire unconditionally after 60 minutes.)
  • This information will not be shared with any third parties.
  • There is no tracking software (Google Analytics, et. al) embedded in this application, and no visitor data is collected for this purpose. There are no cookies used by Super Tuner, and no third-party cookies are used to collect information about you.
